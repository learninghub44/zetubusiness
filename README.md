# Zetu Business Solutions Website

A modern, professional website for Zetu Business Solutions - a software company specializing in web development, mobile apps, cloud infrastructure, and enterprise systems.

## Features

- **Responsive Design**: Mobile-first approach with beautiful UI
- **Modern Technology Stack**: HTML5, CSS3, and vanilla JavaScript
- **Multiple Sections**: Services, Solutions, Industries, Portfolio, Pricing, Team, Testimonials
- **Media Support**: Videos and images gallery
- **Smooth Animations**: Interactive elements with smooth transitions
- **Contact Form**: Direct communication channel for clients
- **SEO Optimized**: Proper meta tags and semantic HTML

## Getting Started

### Prerequisites

- Python 3.x (for the built-in HTTP server)
- Modern web browser

### Installation

1. Clone the repository or download the files
2. Navigate to the project directory
3. Run the development server:

```bash
npm run dev
```

The website will be available at `http://localhost:8000`

### Alternative: Using Live Server

For live reload functionality:

```bash
npm install
npm run serve
```

## Adding Media Files

### Images
Place your portfolio and team images in the `assets/images/` directory:
- `project1.jpg`, `project2.jpg`, `project3.jpg` - Portfolio projects
- `team1.jpg`, `team2.jpg`, etc. - Team member photos

### Videos
Place your demo and promotional videos in the `assets/videos/` directory:
- `hero-video.mp4` - Hero section background video
- `demo1.mp4`, `demo2.mp4`, `demo3.mp4` - Gallery videos

## Customization

### Colors
The main accent color is `#00d4ff` (Cyan). To change:
1. Open `style.css`
2. Replace `#00d4ff` with your desired color
3. Update `#050816` (dark background) if needed

### Company Information
Edit the following in `index.html`:
- Company name and tagline
- Contact information
- Pricing plans
- Team members

### JavaScript Functionality
Edit `app.js` to customize:
- Contact form handling
- Navigation active states
- Scroll animations
- Counter functionality
